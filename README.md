# gravitational-wave-chirp-or-the-one-you-chose-
Binary black hole inspiral gravitational wave chirp simulation

##Physics Background
Gravitational waves emitted by a binary system increase in frequency as orbital separation decreases. The chirp frequency evolves as: f(t) ∝ (tc − t)^(-3/8) where: * f(t) = gravitational wave frequency * tc = coalescence time * t = observation timeThis behavior is derived from: * energy loss via gravitational radiation * quadrupole approximation * Keplerian orbital evolution
